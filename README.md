# 🐍 Python Snake Game
A classic Snake Game built using Python and Turtle Graphics.

This project was created while learning Object-Oriented Programming (OOP) in Python.

## Features

- 🎮 Press Space to Start
- ⏸️ Pause and Resume using P
- 🔄 Restart Game using R
- 🚪 Exit Game using Esc
- ❤️ "Thanks for Playing!" exit screen
- 🍎 Food Collection
- 📈 Live Score Counter
- 🐍 Snake Growth
- 💥 Game Over on Wall or Tail Collision
- 🏆 Persistent High Score using File Handling

## Technologies Used

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)
- Git
- GitHub

## Controls

- ⬆️ Up Arrow - Move Up
- ⬇️ Down Arrow - Move Down
- ⬅️ Left Arrow - Move Left
- ➡️ Right Arrow - Move Right
- 🎮 Space - Start Game
- ⏸️ P - Pause / Resume
- 🔄 R - Restart Game
- ❌ Esc - Exit Game
